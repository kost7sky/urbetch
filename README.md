# Лендинг Urbetch burguês

Эта папка содержит статическую посадочную страницу для бразильской кампании книги `Urbetch burguês`.
Страницу можно открыть локально в браузере или опубликовать как простой HTML-сайт.

## Файлы

- `index.html` - основная страница.
- `trecho.html` - локальный резервный HTML-фрагмент для чтения/экспорта в PDF; в основном лендинге CTA фрагмента ведет на страницу издательства.
- `styles.css` - адаптивные стили.
- `assets/cover-brazil-a5-tight.jpg` - основная A5-версия бразильской обложки для лендинга.
- `assets/cover-brazil.jpg` и `assets/cover.jpg` - исходная story-версия бразильской обложки.
- `assets/cover-russian.jpg` - обложка русской версии; сейчас не выводится поверх страницы, чтобы не портить визуал.
- `assets/breakfast.png`, `conversation.png`, `thinking.png` - иллюстрации кампании.
- В подписи к бразильской обложке добавлена ссылка на русскую версию на Ridero.
- В футере добавлены публичные ссылки автора: Taplink, видео и e-mail.

## Как использовать сейчас

1. Визуальная проверка `index.html` считается пройденной по допущению пользователя от 2026-07-07.
2. Три читательских пути подключены: фрагмент в Google Drive, покупка e-book и покупка печатной книги на сайте Editora Imagina.
3. Кнопки заявок ведут на Google Form кампании.
4. Следующий рабочий шаг: опубликовать всю папку `landing_page` целиком на собственном сервере или другом статическом хостинге.

## Варианты публикации

Вариант на собственном сервере:

1. Скопировать содержимое `landing_page/` в публичную директорию сайта.
2. Настроить `index.html` как стартовую страницу.
3. Проверить, что `styles.css`, `trecho.html` и `assets/*` отдаются по относительным путям.
4. Подключить HTTPS и домен.
5. После публикации проверить внешние CTA: Google Drive, Google Form, Editora Imagina, Ridero, Taplink, YouTube.

Вариант через Netlify/Vercel:

1. Создать аккаунт в Netlify или Vercel.
2. Загрузить папку `landing_page` как статический сайт.
3. Подключить домен позже, когда страница будет вычитана и связана с реальными ссылками.

Вариант без кода:

1. Пересобрать ту же структуру в Tilda, Carrd, Wix или другой платформе.
2. Использовать тексты из `index.html` как основу публичного copy.
3. Использовать изображения из папки `assets`.

## Что нужно решить перед публикацией

- Ссылка покупки e-book: подключена страница продукта Editora Imagina.
- Ссылка покупки печатной книги: подключена отдельная страница продукта Editora Imagina.
- Ссылка на бесплатный фрагмент: подключен Google Drive-файл с открытым доступом.
- Канал сбора заявок: подключена Google Form кампании.
- Контакт человека или партнера в Natal, который будет получать заявки на встречи.
- Короткое видео автора: ссылка добавлена в футер; для кафе можно отдельно подготовить Reels/вертикальную версию с PT-BR субтитрами.

Подробная инструкция по подключению ссылок вынесена в `../task_05_connect_landing_links_ru.md`.
